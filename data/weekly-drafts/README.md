# Weekly Drafts

This directory stores AI-assisted weekly issue drafts.

Drafts are not public output until reviewed and published with:

```powershell
npx pnpm weekly:publish -- --week YYYY-Www --file data/weekly-drafts/YYYY-Www.ai-draft.json
```

Weekly drafts may contain weak, promotional, overly technical, or insufficiently
verified items. Open original links before publishing. Video leads must not be
published from a draft unless the maintainer has watched the source and checked
date, place, action, result, privacy risk, and source trail.
