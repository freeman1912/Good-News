# 深圳市恒晖公益基金会

## Why Watch

深圳市恒晖公益基金会由陈行甲发起，公开资料显示其关注大病救助、公益卫生技术评估、乡村医生培训、青少年心理健康和教育关怀等方向。它适合进入长期公益组织观察池，但不应因为个人知名度自动收录。

## Suitable Good-News Angles

- 已执行的大病救助项目，有患者援助机制和实际受益人数。
- 青少年心理健康、教育关怀项目的阶段性成果。
- 乡村医生培训项目的培训人数、覆盖地区、后续服务结果。
- 年报中可核实、可追踪的项目成果。

## Risks

- 个人光环叙事大于项目结果。
- 基金会薪酬、治理、透明度争议需要如实避让或注明，不适合做无条件赞美。
- 医疗/救助项目涉及患者隐私，不公开具体个人敏感信息。
- 企业、基金会或个人单方表述需要交叉核实。

## Review Rule

Publish project results, not personality profiles. A Henghui-related item should
enter the main timeline only when it has visible beneficiary outcomes, accessible
project information, and a privacy-safe source trail.
